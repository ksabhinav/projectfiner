------------------------------------------------------------------------------
GRANULARITY FINDING — IMPORTANT
------------------------------------------------------------------------------

Punjab SLBC publishes most quarterly data BANK-WISE (rows = banks across all
Punjab), not DISTRICT-WISE. This is the OPPOSITE of Haryana / HP / Delhi
(all on the same PNB-hosted CMS) where the basic-banking-data annexure is
district-wise.

Verified by direct file inspection:

  Annexure-3 / -16 / -20 "BASIC BANKING DATA" (every quarter):
      Sheet: "BANK WISE"  -> rows = 30+ banks, columns = Branches /
      Deposits / Advances / CD Ratio (Rural/Semi-Urban/Urban/Total).
      Sample: Annexure-16-BASIC-BANKING-DATA-JUNE25.xlsx -> 44 rows
      with column "Bank Name" as primary key. NO district column.

  Annexure-*-AREAWISE-CD-RATIO (every quarter):
      Aggregate state CD ratio split by Rural / Semi-Urban / Urban
      branches. NOT a per-district split.

  Annexure-*-Jalandhar-CD-Ratio / Annexure-*-Kapurthala-CD-Ratio /
  Annexure-*-SBS-Nagar-CD-Ratio / Annexure-*-Hoshiarpur-CD-Ratio /
  Annexure-*-Pathankot-CD-Ratio / Annexure-*-Rupanagar-CD-Ratio:
      Per-district CD ratio mini-tables for 6 Doaba-region districts
      only (Jalandhar, Kapurthala, SBS Nagar, Hoshiarpur, Pathankot,
      Rupnagar). Inside each: bank-wise rows for that district. Not
      a Punjab-wide table.

What IS district-wise (all 22/23 districts in one table):

  Annexure-*-ACP-Districtwise (every quarter Sep-2020 onwards):
      "DISTRICT WISE ACHIEVEMENTS VIS A VIS TARGETS UNDER ANNUAL
      CREDIT PLAN" -- 4 buckets x 3 metrics per district:
        Agri / MSME / Other PS / Total PS  ->  Target, Achievement,
        % Achievement.  Amounts in Crores.  Currently the only
        consistently-quarterly district-wise sheet across the
        full meeting archive.

  Annexure-*-NRLM-Districtwise (sporadic):
      SHG / NRLM credit linkage by district. Available in some 2022
      meetings but not consistently every quarter.

  Annexure-13.1-District-wise-Ghar-Ghar-KCC (sporadic, 2024+):
      KCC saturation campaign progress per district. Single quarter.

  Annexure-5/-6-KCC-District-wise (sporadic):
      KCC daily monitoring per district. Single-shot annexures.

Conclusion: Punjab is a PRIORITY-SECTOR-DISTRICT-WISE state in FINER (the
analogue of Madhya Pradesh which is CD-ratio-district-wise-only). Punjab
contributes:

  priority_sector__agri_target_amt / _achievement_amt / _achievement_pct
  priority_sector__msme_target_amt / _achievement_amt / _achievement_pct
  priority_sector__ops_target_amt  / _achievement_amt / _achievement_pct
  priority_sector__total_ps_target_amt / _achievement_amt / _achievement_pct

per district per quarter.

Punjab does NOT contribute district-wise CD ratio, deposits, advances, branch
counts, PMJDY, MUDRA, or SHG to FINER -- those metrics are state-aggregate
only in the source. The standard FINER FI indicators (credit_deposit_ratio,
branch_network, pmjdy, kcc) will show Punjab as a blank state in the choropleth.

------------------------------------------------------------------------------
UNIT CONVERSION
------------------------------------------------------------------------------

Source unit: Crores (header literally says "Amount in cr").
FINER unit:  Lakhs.
Applied:     monetary fields (Target_amt, Achievement_amt) multiplied by 100.
             Achievement_pct kept as is (source uses 0.92 decimal form ->
             converted to 92 % to match FINER convention).

Sample Ludhiana priority_sector__total_ps_achievement_amt:
  Dec 2024 source: Rs 40,321.06 Crores  -> stored as 4,032,106.10 Lakhs
  Dec 2025 source: Rs 49,631.88 Crores  -> stored as 4,963,188.12 Lakhs
